Welcome to reallysimplesoftware's documentation!

What's this ?
-------------

This is a RSS reader that have three primary goals :

  * Allow you to read your RSS threads, including those which only
    propose introductions to full articles !
  * Allow you to store those you want to keep
  * Allow you to watch videos inside the RSS reader
